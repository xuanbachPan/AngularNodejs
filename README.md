# AngularNodejs
Front-end: Angular version: 20, add lib @angular/material (commands: cd request-app; ng add @angular/material; ng serve)
Back-end: Node.js version: 24 (commands: cd expense-rest-api; npm run start || node server.js)
Database: SQLite (commands: cd expense-rest-api; npm init; npm install express sqlite3 cors )

This simple project bases on examples of Tutorialspoint.
link: https://www.tutorialspoint.com/angular/index.htm 
(chapter Angular HTTP Client programming and others...)
